A simple plugin to Filter, edit or Modify Image.

== Description ==

A simple editing Image tool. Just upload your image and click the button to edit, modify or Filter your image.
	* [LIVE DEMO](https://kovatz.com/image-filter-plugin/)
	* [About Author](https://kovatz.com)

== Installation ==

1. Upload `image-filter.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place <strong>[kov_image_filter]</strong> shortcode in your page or post.


== Frequently Asked Questions ==

= Will this work in Gutenberg or Elementor? =

It will work in every framework including Gutenberg, Elementor, Wp-Bakery or Classic editor
